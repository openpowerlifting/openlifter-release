Perguntas Frequentes

  • O OpenLifter tem suporte para libras?

    Sim.

  • Um competidor pode estar em várias divisões ou eventos?

    Sim. Na página Cadastro, os campos para a Divisão e o Evento aceitam múltiplos valores para um competidor. A primeira Divisão inserida será a mostrada na tela da tabela de Levantamento.

  • Posso ter múltiplas plataformas?

    Sim. Você irá precisar de alguma forma de transferir arquivos entre computadores, como um dispositivo USB. Configure a competição e cadastre todos os competidores em um computador, então Salve na Tela Inicial. Transfira este arquivo salvo para os computadores com outras plataformas e Carregue-o no OpenLifter. Depois de terminado o levantamento, a tela de Resultados terá os botões de Exportar e Mesclar para cada plataforma. Use o dispositivo USB para mesclar os arquivos exportados. Então você terá todos os resultados em um computador e estes resultados incluirão todas as plataformas.

  • Se eu alterar o cadastro após o início do levantamento, vai explodir tudo?

    Não, nada vai explodir não. O OpenLifter permite a alteração de qualquer coisa a qualquer momento e ele fará a coisa certa.

  • Como faço para marcar as alturas de segurança do supino? E quanto ao rack agachamento in/out?

    Na página Pesagens, as entradas do "Rack Agachamento" e "Rack Supino" aceitam textos livres, não apenas números. No geral, uma altura de 5 de rack supino com um peso de segurança de 3 é escrito como "5/3", mas você pode usar qualquer notação que sua equipe de carregamento preferir. Da mesma forma, para o rack agachamento, você pode digitar "15in".

  • Como faço para corrigir um levantamento marcado incorretamente?

    Use os controles no canto inferior esquerdo da página Levantamento para selecionar novamente o levantador e, então, clique no botão Inválido ou Válido para alterar.

  • Como altero a altura do rack? E quanto ao peso inicial do próximo grupo?

    Na página Levantamento, clique no botão "Alternar Pesagens". Isso incorporará a página Pesagens à página Levantamento para que você possa fazer estas alterações e manter o carregamento projetado. Quando tiver terminado de fazer alterações, clique novamente no botão "Alternar Pesagens" para voltar.

  • Com que frequência eu devo salvar para o arquivo?

    Uma boa regra a seguir é de salvar em arquivo antes de fechar o OpenLifter.

    O OpenLifter salva automaticamente na memória do navegador após cada alteração. Se você fechar o navegador acidentalmente, tudo continuará lá quando você o reabrir. Mas se você não usar o OpenLifter por alguns dias, navegadores podem esquecer as coisas, então sua competição pode ser esquecida (apesar de ser improvável). Se quiser garantir que sua competição ainda esteja lá dentro de alguns dias, salve em arquivo. Não dói.

  • O texto na página Levantamento é grande demais!

    Use os botões de Zoom do seu navegador para ajustar o conteúdo à sua tela. A maioria dos navegadores também permite alterar as configurações de zoom segurando Ctrl e rolando a roda do mouse.

  • O texto na página Resultados é pequeno demais!

    Use os mesmos recursos de "Mais Zoom" e "Menos Zoom" para ajustar as coisas. Se você deixar o texto bem grande, poderá ajustar à seu projetor e fazer uma apresentação de slides improvisada no final da competição. O público parace gostar disso porque lhes dá algo para olhar.

  • Posso importar automaticamente detalhes do levantador na página de cadastro?

    Sim, exportando seus dados em um formato de importação especial CSV. No topo da página Cadastro há um botão para baixar um modelo para que você possa aprender, por exemplo, como esse formato é. Se o formato não corresponder ao que esperamos, será apresentada uma mensagem de error útil. Uma vez aprendido (através de tentativa e erro) o que o programa espera, a próxima vez será bem mais fácil.

  • O que são números de Grupo?

    Números de Grupo são uma forma de decidir quem vai primeiro quando dois levantadores pedem o mesmo peso. O levantador com o número de grupo menor vai primeiro. Algumas federações exigem o uso deste sistema. Geralmente os números de grupo são atribuídos pelo diretor da competição baseado na ordem de cadastramento.

    O uso de números de grupo é completamente opcional. Se você não usa números de grupo, o OpenLifter irá decidir empates mantendo a mesma ordem relativa da tentativa anterior. Isto funciona muito bem e é o esperado pelos levantadores. Se sua federação não exige números de grupo, recomendamos não usá-los.

  • Existe uma ficha de pontuação online?

    Não existe. Uma vez que o OpenLifter é iniciado, ele é inteiramente offline. Ele roda no seu computador, não no nosso servidor.

  • Vejo classes de peso H, M, Mx. O que é Mx?

    Mx (pronunciado Muks é um título de gênero neutro — assim como Sr. e Sra. — originário do Reino Unido.

    A maioria das federações exige que os levantadores compitam nas categorias masculina ou feminina. Algumas federações permitem que os levantadores entrem em uma terceira categoria. Mx é tudo junto e misturado. Pode incluir levantadores não-binários, sem forçar atribuição à dicotomia padrão.

    Se você quiser fazer uma competição onde homens e mulheres competem juntos, você pode colocar todo mundo na categoria Mx.

  • Isto envia automaticamente os resultados para a OpenPowerlifting?

    Não. A OpernPowerlifting exige que a federação sancionadora certifique os resultados, assim não podendo serem recebidos diretamente. No entanto, se você clicar no botão "Exportar para OpenPowerlifting" na página Resultados, os resultados serão baixados em um formato que é muito conveniente para nós. Se você quiser faciliar nossas vidas, enviar esses arquivos seria bem útil. Podemos adicioná-los após o órgão sancionador aprová-los.

  • Como eu configuro as telas para a sala de aquecimento?

    Use um programa de compartilhamento de área de trabalho para sua rede local e, então, compartilhe a tela. Programas de bate-papo com recursos de compartilhamento de tela também servem.
