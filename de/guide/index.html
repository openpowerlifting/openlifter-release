FAQ

  • Unterstützt OpenLifter Pfund (lbs)?

    Ja.

  • Kann ein Teilnehmer in mehreren Klassen oder Disziplinen eingetragen sein?

    Ja. Auf der Registrierungsseite können für einen Teilnehmer mehrere Klassen und Disziplinen eingetragen werden. Die erste Disziplin die eingetragen wird, wird auf der angezeigten Wettkampfstabelle erscheinen.

  • Kann es mehrere Wettkampfplattformen geben?

    Ja. In diesem Fall muss eine Möglichkeit gefunden werden die Daten zwischen mehreren Computern auszutauschen, üblicherweise per USB. Erstellen Sie den Wettkampf und tragen Sie alle Teilnehmer auf einem Computer ein, dann speichern Sie auf der Startseite. Übertragen Sie dann die gespeicherte Datei auf alle Computer der einzelnen Wettkampfplattformen und laden Sie die Datei in OpenLifter hoch. Sobald die Veranstaltung beended ist, erscheinen auf der Ergebnisseite Buttons zum exportieren und zusammenfügen aller Plattformen. Nutzen Sie den USB Stecker zum zusammenfügen und exportieren. Dann haben Sie alle Ergebnisse auf einem Computer und alle Plattformen sind in den Ergebnissen enthalten.

  • Geht alles kaputt wenn ich nach dem Beginn des Hebens was an der Registrierung ändere?

    Nichts geht kaputt. Man kann alles zu jeder Zeit in OpenLifter ändern und die Software funktioniert weiterhin richtig.

  • Wie markiere ich die Bankdrücken-Sicherheitshöhen? Was ist mit Kniebeuge-Gestellt innen/außen?

    Die Felder für die Kniebeuge und Bankdrücken auf der Seite zum Wiegen erlauben freiform Text, nicht nur Zahlen. In dem meisten Fällen wird eine Bank mit der Höhe 5 und der Safeties 3 als "5/3" geschrieben, Sie können aber eine beliebige Schreibweise wählen, je nach dem was Gang und Gebe ist. Verfahren Sie genauso bei der Höhe für die Kniebeuge, z.B. "15cm".

  • Wie ändere ich einen Versuch der als falsch markiert ist?

    Nutzen Sie die Steuerung in der Ecke unten links auf der Veranstaltungsseite um den Teilnehmer abermals auszuwählen. Dann klicken Sie auf den ungültig oder gültig Button um den Wert zu ändern.

  • Wie ändere ich die Rack-Höhe? Wie ändere ich Startgewichte im nächsten Durchlauf?

    Auf der Wettkampfseite gehen sie auf den Button "Körpergewichte ändern". Diese Aktion öffnet die Seite mit den Gewichten auf der Wettkampfseite, sodass Sie alle diese Einstellungen ändern können während die Wettkampfseite weiterhin normal angezeigt wird. Wenn Sie alle Änderungen vorgenommen haben, klicken Sie abermals auf den Button "Körpergewicht ändern" um das Fenster zu schließen.

  • Wie oft sollte ich die Datei speichern?

    Als generelle Empfehlung sollten sie die Datei jedes mal vor dem Schließen von OpenLifter speichern.

    OpenLifter speichert alle Änderungen automatisch im Browserverlauf. Falls Sie versehentlich den Browser schließen, sollten alle Details beim Öffnen noch vorhanden sein. Falls Sie aber OpenLifter für ein paar Tage nicht benutzen, kann es sein, dass der Browser Ihren Wettkampf vergisst (obwohl das unwahrscheinlich ist). Wenn Sie also sichergehen wollen, dass Ihr Wettkampf vorhanden ist, speichern Sie die Datei auf dem Computer ab. Es kann nicht schaden.

  • Die Schriftgröße auf der Wettkampfseite ist viel zu groß!

    Nutzen Sie die "Zoom in" und "Zoom out" buttons Ihres Browsers um die Schriftgröße an ihren Bildschirm anzupassen. In den meisten Browsern lässt sich die Größe auch mit drücken der Strg Taste und dem Mausrädchen regeln.

  • Die Schriftgröße auf der Ergebnisseite ist viel zu klein!

    Nutzen Sie die gleichen "Zoom in" und "Zoom out" Funktionen Ihres Browsers um die Größe anzupassen. Wenn Sie den Text sehr groß machen können Sie ihn auf die volle Größe des Beamers anpassen und in eine spontane Slideshow umwandeln. Das Publikum scheint diese Funktion zu mögen, da es ihnen etwas zum angucken bietet.

  • Lassen sich Teilnehmerdaten automatisch auf der Registrierungsseite hochladen?

    Ja, indem Sie die Daten in ein spezielles CSV Format exportieren. Oben auf der Registrierungsseite ist ein Button zum herunterladen von Tabellen, der Ihnen anhand eines Beispiels zeigt, wie das Format auszuschauen hat. Falls das Format nicht mit unserem Beispiel übereinstimmt, wird Ihnen eine hilfreiche Fehlermeldung angezeigt. Sobald Sie gelernt haben wie das Format auszusehen hat (trial and error), wird der Import beim nächsten mal ein Kinderspiel sein.

  • Wofür gibt es dei Teilnehmer-Nummern

    Die Teilnehmer-Nummern helfen zu Entscheiden wer zuerst antritt wenn zwei Teilnehmer mit dem gleichen Gewicht starten. Der Teilnehmer mit der niedrigeren Nummer startet zuerst. Manche Verbände schreiben dieses Vorgehen vor. Normalerweise wird die Reihenfolge der Nummern vom Organisator in der Riehenfolge der Registrierungen vergeben.

    Die Nutzung der Teilnehmer-Nummern ist nicht dringend nötig. Wenn Sie keine Nummern vergeben, wird OpenLifter bei gleichem Gewicht die gleiche Reihenfolge wie im vorherigen Versuch nutzen. Diese Vorgehensweise funktioniert gut und wird von den Teilnehmern erwartet. Wenn Ihr Verband keine Teilnehmer-Nummern vorschreibt empfehlen wir diese nicht zu nutzen.

  • Gibt es eine Online-Ergebnisliste?

    Leider nein. Sobald Sie OpenLifter geöffnet haben, läuft das Programm komplett offline. Es speichert auf Ihrem Computer, nicht auf unserem Server.

  • Ich sehe M, W, und Mx Gewichtsklassen. Was ist Mx?

    Mx (Muks ausgesprochen) ist eine geschlechtsneutrale Ansprache — wie z.B. Mr. und Ms. — die in England enstanden ist.

    Die meisten Verbände schreiben vor, dass Teilnehmer in den Klassen männlich und weiblich antreten müssen. Manche Verbände erlauben es Teilnehmern in einer dritten Klasse anzutreten. Mx ist inklusiv. In der Klasse können nicht-binäre Teilnehmer eingetragen werden ohne Sie auf die Standard-Zweispaltung zu beschränken.

    Wenn Sie einen Wettkampf veranstalten auf dem Männer und Frauen alle zusammen in einer Klasse antreten, können Sie alle Teilnehmer in die Mx Klasse eintragen.

  • Werden die Ergebnisse automatisch auf OpenPowerlifting hochgeladen?

    Nein. Damit Ergebnisse auf OpenPowerlifting erscheinen können, müssen sie zuerst von dem Verband zertifiziert werden. Ergebnisse können also nicht direkt weitergeleitet werden. Wenn Sie aber den "Export for OpenPowerlifting" button auf der Ergebnisseite klicken, dann werden die Ergebnisse in einem Format das sehr hilfreich für uns ist heruntergeladen. Wenn Sie unsere Arbeit also erleichtern wollen schicken Sie uns diese Datein. Wir können sie dann leichter hochladen nachdem Sie von dem Verband bestätigt wurden.

  • Wie richte ich Monitore im Aufwärmbereich ein?

    Nutzen Sie Software zum Teilen von Bildschirmen in Ihrem Netzwerk und teilen Sie den Bildschirm. Bildschirm-Teilen-Funktionen in Chat-Programmen funktionieren auch.
